Je veux créer un site web professionnel complet pour AVERNA,
entreprise de collecte de déchets ménagers à Brazzaville, Congo.

## STRUCTURE DU PROJET
Crée un dossier "site-averna" avec cette organisation :
site-averna/
  index.html          (page d'accueil vitrine)
  inscription.html    (formulaire d'abonnement)
  admin.html          (tableau de bord admin)
  css/style.css
  js/main.js
  js/admin.js
  backend/server.js   (Node.js + Express)
  backend/db.json     (stockage des inscriptions)
  images/             (dossier pour les images)
  package.json

## PAGE 2 : inscription.html (formulaire)
Formulaire complet avec :
- Nom, Prénom (côte à côte)
- Téléphone, WhatsApp (côte à côte)
- Quartier/Arrondissement (liste déroulante 8 arrondissements)
- Adresse précise / repère
- Type de client : Particulier / Commerce / Entreprise / Résidence
- Taille du bac : 50L / 100L / 240L (sélection visuelle avec cartes)
- Fréquence : 2 / 3 / 4 / 6 passages par semaine
- Message libre (facultatif)
- Case à cocher consentement RGPD
- Bouton "Envoyer ma demande"
Sidebar avec : étapes du processus, zones desservies, contacts

## PAGE 3 : admin.html (tableau de bord)
- Login simple par mot de passe (mot de passe : averna2025)
- Tableau listant tous les inscrits avec colonnes :
  Date | Nom complet | Téléphone | Quartier | Bac | Fréquence | Statut
- Bouton "Exporter CSV" pour télécharger la liste
- Compteur total d'inscrits
- Filtre par quartier et par statut
- Bouton pour changer le statut (En attente / Contacté / Abonné)

## BACKEND : backend/server.js
- Framework : Express.js
- Port : 3000
- Routes :
  POST /api/inscription → enregistre dans db.json
  GET /api/inscriptions → retourne tous les inscrits (protégé)
  PUT /api/inscription/:id/statut → met à jour le statut
  GET /api/export-csv → télécharge un CSV de tous les inscrits
- Validation des champs obligatoires
- CORS activé
- db.json initialisé avec tableau vide []

## PACKAGE.JSON
Dépendances : express, cors, body-parser
Script start : "node backend/server.js"

1. Crée tous les fichiers
2. Installe les dépendances avec npm install
3. Lance le serveur avec node backend/server.js

5. Le site doit être 100% en français

7. Utilise les vraies images depuis le dossier images/
